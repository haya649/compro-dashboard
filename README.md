# 競プロ学習ダッシュボード

## 概要
AtCoder IDを入力すると、そのユーザーの学習状況（AC数、レート、提出割合など）をグラフで可視化するWebアプリケーションです。

## 主な機能
- **メインページ**
  - AC（正解）数、AC数ランク
  - 現在のレート、最高レート
  - 最新のAC履歴（5件）
- **グラフページ**
  - 提出結果の割合（円グラフ）
  - 日別のAC数（折れ線グラフ）
  - レート変動（折れ線グラフ）
  - Ratedコンテストの参加回数

## 技術スタック
- **言語:** Python
- **フレームワーク:** Streamlit
- **データ連携:** requests (AtCoder Problems API, AtCoder公式/json)
- **データ処理:** pandas
- **グラフ描画:** Plotly, Streamlit標準グラフ
